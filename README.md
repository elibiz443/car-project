# CAR-PROJECT (TESTING IN RSPEC)
These are ruby codes that have been tested in rspec.

Geting Started:
* Clone the repo.
* Run: bundle

Requirements:
Add:
```
gem "rspec"
```
bundle

Run:

```
bundle exec rspec spec --init
```
In .rspec file add:
```
--color
--format documentation
```

Run tests:
```
bundle exec rspec
```

Help:
```
bundle exec rspec --help
```
